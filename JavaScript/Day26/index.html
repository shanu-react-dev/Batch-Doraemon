JavaScript

JavaScript is a High Level Programming Language as well as client side scripting language. It is used to add functionalities to our static web pages or websites.

History

It was introduced in 1995 by Brendan Eich at Sun Microsystem. The first name of JavaScript is Mocha later it got changed as LiveScript and again for its marketing purpose it got renamed as JavaScript. Later in 1996 it got one competitor from Microsoft as JScript. To analyze which one is better we got one organization as ECMA (European Computer Manufacturer Association). Then in 1996 it got renamed as ECMAScript. From that onwards ECMA is responsible for providing updates.

Features of JavaScript

JavaScript comes up with multiple features which enhances its performance and makes it different from other programming languages. Below it is listed with Explanations.

  1. High Level Programming language
    High Level Programming language dictates human understandable language.
  2. Client Side Scripting Language
    A programming language which can run on browser or no need of making server request to update the UI is known as Client Side Scripting Language
  3. Interpreted
    Interpreted represents line by line execution of the codes.
  4. Dynamically Typed
    No need to specify the datatype while declaring the variable. Datatypes will be checked at the runtime.
  5. Weakly Typed
    We can assign any value to the same variable in JavaScript. It is not statically typed.
  6. Loosely Typed
    It does not have restricted syntax. No need to specify semicolon.
